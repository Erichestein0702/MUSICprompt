## 🚀 快速开始

### 1. 浏览精校合集
直接查看我们精心优化的顶级提示词：
```bash
# 查看精校总索引
cat data/final_output/curated/README.md

# 查看特定流派
cat data/final_output/curated/rock_curated.md
```

### 2. 按场景查找
根据你的使用场景快速定位：
- 🎉 [派对聚会](data/final_output/use_cases/party.md) - 79条高能量提示词
- 📚 [学习专注](data/final_output/use_cases/study.md) - 35条氛围音乐
- 🎮 [游戏电竞](data/final_output/use_cases/gaming.md) - 18条史诗配乐

### 3. 使用提示词
每个精校提示词包含：
- 📝 英文原文 (可直接复制到 Suno/Udio)
- 🈯 中文翻译 (便于理解)
- 💡 使用建议 (场景、音量等)
- ⚙️ 参数调整指南 (BPM、调性、乐器替换)
- 🎨 类似风格推荐

## 📝 使用指南

### 提示词质量评分体系

每个提示词都经过5维度质量评估：

| 维度 | 权重 | 说明 |
|------|------|------|
| 🔧 技术参数 | 25% | BPM、调性、乐器指定 |
| 🏗️ 结构清晰度 | 20% | [Verse]、[Chorus]等标签使用 |
| 🎵 流派明确度 | 20% | 具体流派描述 |
| 🎸 乐器精确度 | 20% | 具体乐器名称 |
| 📏 长度适当性 | 15% | 50-950字符范围内 |

### 精校提示词特色

⭐ **精校版提示词** (8分以上) 额外包含：
- 💡 3条实用使用建议
- ⚙️ 参数调整指南 (BPM±10、调性变化等)
- 🎨 3个类似风格推荐
- 🎯 适用场景标注

## 🤝 贡献指南

欢迎提交高质量的AI音乐提示词！请确保：
1. 提示词包含明确的技术参数 (BPM、调性)
2. 指定具体乐器 (如 "crunchy distorted guitar" 而非 "guitar")
3. 使用标准结构标签 ([Verse], [Chorus]等)
4. 长度控制在 50-950 字符
